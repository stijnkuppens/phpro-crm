```bash
npx shadcn@latest add chart
```

3. Add `react-is` override to package.json for React 19 compatibility with recharts. Add an `"overrides"` key at the top level of package.json:
```json
{
  "overrides": {
    "react-is": "$react"
  }
}
```
If an `"overrides"` key already exists, add `"react-is": "$react"` to it.

4. Run `npm install` again to apply the override.

5. Verify recharts is v2.x (not v3.x):
```bash
npm ls recharts
```
