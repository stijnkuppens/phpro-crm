# DAM Page Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the flat file list page with a full Digital Asset Management UI featuring folder tree, grid/list views, thumbnails, detail panel, and bulk operations.

**Architecture:** Pure Supabase Storage — folders are path prefixes in the existing `documents` bucket. No DB migrations. Central `useFileBrowser` hook manages all DAM state. Components compose around it.

**Tech Stack:** Next.js App Router, React 19, Supabase Storage, shadcn/ui (base-ui), Tailwind CSS v4, Lucide icons

```ts
// src/features/files/actions/delete-file.ts
'use server';

import { requirePermission } from '@/lib/require-permission';
import { createServiceRoleClient } from '@/lib/supabase/admin';
import { logAction } from '@/features/audit/actions/log-action';

export async function deleteFile(path: string) {
  await requirePermission('files.delete');
  const admin = createServiceRoleClient();
  const { error } = await admin.storage.from('documents').remove([path]);
  if (error) throw new Error(error.message);

  await logAction({
    action: 'file.deleted',
    entityType: 'file',
    metadata: { path },
  });
}

export async function deleteFiles(paths: string[]) {
  await requirePermission('files.delete');
  const admin = createServiceRoleClient();
  const { error } = await admin.storage.from('documents').remove(paths);
  if (error) throw new Error(error.message);

  await logAction({
    action: 'file.deleted',
    entityType: 'file',
    metadata: { paths },
  });
}
```

And change `upsert: false` to `upsert: true`.

### Task 4: Create useFileBrowser hook

**Files:**
- Create: `src/features/files/hooks/use-file-browser.ts`

- [ ] **Step 1: Implement the central DAM state hook**

The hook manages:
- `currentPath` (string, starts as `''`)
- `navigateTo(path)` — sets currentPath, clears selection and active file
- `breadcrumbs` — derived from currentPath split on `/`
- `files` / `folders` — fetched via `supabase.storage.from('documents').list(currentPath, { search })`, split by `id === null`
- `loading` state
- `search` / `setSearch` — debounced, triggers re-fetch
- `selectedFiles` (Set<string>) — toggle, selectAll, clearSelection
- `activeFile` / `setActiveFile` — for detail panel
- `viewMode` ('grid' | 'list') — persisted in state
- `deleteFiles(names)` — calls the `deleteFiles` server action with full paths, then refreshes
- `createFolder(name)` — uploads `.emptyFolderPlaceholder` to `{currentPath}/{name}/`, then refreshes
- `refresh()` — re-fetches current folder

Use `useEffect` to fetch when `currentPath` or `search` changes.
